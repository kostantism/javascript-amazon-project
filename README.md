# Amazon Clone Project

A simple fully functional, responsive E-commerce web application built with JavaScript. This project replicates the core user experience of Amazon, featuring a dynamic product catalog, advanced cart logic, a multi-step checkout process, and automated testing.

> **Credits:** This project was built by following the JavaScript course created by [SuperSimpleDev](https://www.youtube.com/@SuperSimpleDev).

---

## Features

### Dynamic Product Catalog
- Products are fetched from a backend API.
- Dynamic rendering using JavaScript classes.

### Advanced Cart Management
- Add and remove items from the cart.
- Update product quantities in real time.
- Automatic calculation of cart badges and order subtotals.
- Persistent cart storage using localStorage.

### Interactive Checkout
- Integrated delivery date selection using **Day.js**.
- Automatically skips weekends for delivery scheduling.
- Real-time payment summary updates.
- Includes shipping costs and estimated tax calculations.

### Order System
- Submit orders to a practice backend API.
- View order history.
- Track package deliveries.

### Responsive UI
- Fully optimized for:
  - Mobile
  - Tablet
  - Desktop
- Built using **CSS Grid** and **Flexbox**.

### OOP & Procedural Logic
- Multiple implementations of cart logic:
  - Procedural Programming
  - Object-Oriented Programming (OOP)
  - ES6 Classes
- Demonstrates different architectural patterns.

### Automated Testing
- Comprehensive Unit and Integration testing using the **Jasmine** framework.
